72  * @param[in] g The GDisplay to display this window on
73  * @param[in] gb The GCheckboxObject structure to initialise. If this is NULL, the structure is dynamically allocated.
74  * @param[in] pInit The initialization parameters to use
75  *
76  * @note The drawing color and the background color get set to the current defaults. If you haven't called

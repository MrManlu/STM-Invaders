611  /**
612  * @brief Draw a selection of 45 degree arcs of a circle
613  * @pre GDISP_NEED_ARCSECTORS must be TRUE in your gfxconf.h
614  *
615  * @param[in] g The display to use
616  * @param[in] x,y The center of the circle
617  * @param[in] radius The radius of the circle
618  * @param[in] sectors Bits determine which sectors are drawn.
619  * Bits go anti-clockwise from the 0 degree mark (y = 0, x is positive), as follows:
620  * bit 0 - upper right right -----
621  * bit 1 - upper upper right /2 1\
622  * bit 2 - upper upper left /3 0\
623  * bit 3 - upper left left \4 7/
624  * bit 4 - lower left left \5 6/
625  * bit 5 - lower lower left -----
626  * bit 6 - lower lower right
627  * bit 7 - lower left left
628  * @param[in] color The color to use
629  *
630  * @note This is a more limited versions of the general arc drawing routine. It
631  * doesn't require trig libraries or tables or floating point and is smaller in code size.
632  * There is probably little point in including both this and the general
633  * arc routine as the general arc routine can do everything this can do.
634  *

36  * write() call unless this feature is disabled.
37  * @details If this feature is disabled, the user should sync the filesystem
38  * himself using @p gfileSync()

42  */

106  * @note If GFILE_ALLOW_DEVICESPECIFIC is on then you can ensure that you are

121  * its name with "F|" (the letter 'F', followed by a vertical bar).
122  * @note FATFS and PETITFS offer the same FAT file system support. They just use
